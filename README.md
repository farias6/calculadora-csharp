# Calculadora em C#

Calculadora de console com as 4 operações básicas, histórico de cálculos e tratamento de erros.

## Tecnologias
- C# / .NET 10
- Console App

## Funcionalidades
- Soma, subtração, multiplicação e divisão
- Histórico dos cálculos da sessão
- Tratamento de divisão por zero
- Validação de entrada do usuário

## Como rodar

Pré-requisito: ter o [.NET 10 SDK](https://dotnet.microsoft.com) instalado.

```bash
dotnet run
```

## O que aprendi neste projeto
- Orientação a objetos com classes e responsabilidades separadas
- Tratamento de exceções com try/catch
- Validação de entrada com TryParse
- Coleções com List<T>
- Switch expression (C# 8+)
